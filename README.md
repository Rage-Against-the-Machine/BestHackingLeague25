# Gazetka

This repository contains the source code for the Gazetka project, which is composed of a backend, a web frontend, and a mobile application.

Full documentation in Polish available [here](./docs/dokumentacja.pdf).

## Projects

Below is a brief overview of each component of the project. For more detailed information, please refer to the README file within each project's directory.

*   **[Backend](./backend/README.md)**: The backend server for the Gazetka application, built with Python. It handles business logic, data storage, and provides an API for the clients.

*   **[Frontend](./Frontend/README.md)**: The web application for Gazetka, built using a modern frontend framework. It provides a user-friendly interface for interacting with the application's features.

*   **[Mobile App](./mobile_app/README.md)**: A cross-platform mobile application for Gazetka, developed using Flutter. It offers a native mobile experience for users on both iOS and Android.
